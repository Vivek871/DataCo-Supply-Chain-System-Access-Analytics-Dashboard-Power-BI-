DataCo Supply Chain & Access Logs Analytics Dashboard

Developed an end-to-end Power BI dashboard to analyze supply chain performance and system access logs. The solution provides insights into sales trends, profitability, delivery efficiency, product performance, shipping operations, and department-wise access activities. Implemented interactive slicers, KPI tracking, DAX measures, and advanced visualizations to support data-driven business decisions.
